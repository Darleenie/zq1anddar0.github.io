Amazon Alexa

Alexa credentials are set on the server as environment variables — they never touch the browser. Add these two config vars on Heroku (or your .env file locally) then redeploy:

AMAZON_EMAIL=your@email.com AMAZON_PASSWORD=yourpassword

2FA / OTP: If your Amazon account has two-factor authentication enabled, disable it or create a separate Amazon account for this integration. alexa-remote2 doesn't support interactive 2FA prompts.

Other Platforms (via SmartThings)

Connect these to SmartThings and they'll appear automatically in the device list.

  • Aqara → SmartThings via Matter or Zigbee hub
  • Sengled → SmartThings via Sengled Hub integration
  • Google Nest → SmartThings via Google Home cloud link
